# Videos Directory

This directory contains video files for the CarCompare application.

## Usage

Place your video files here and reference them in the VideoBackground component.

## Recommended Video Specifications

- **Format**: MP4 (H.264)
- **Resolution**: 1920x1080 or higher
- **Duration**: 10-30 seconds (looped)
- **File Size**: Under 10MB for optimal loading
- **Content**: Car showcase, driving scenes, or automotive footage

## Current Files

- `car-showcase.mp4` - Main hero background video (to be added)

## Adding Your Video

1. Place your video file in this directory
2. Update the `videoSrc` prop in the VideoBackground component if needed
3. Ensure the video is optimized for web playback

## Fallback

If no video is available, the component will display a gradient background.
